# 测试用例记录

## 1. 文档说明

本文件用于记录项目的系统化测试用例，采用表格化方式组织，覆盖功能、边界、异常、压力、回归五类测试目标。文档重点体现测试覆盖面、执行规范性和结果可复盘性。

## 2. 测试范围

本轮测试覆盖以下方向：

- 角色控制与移动手感
- Jump 与高低差交互
- 战斗、技能、连击、受击与死亡
- 掉落、拾取、背包、装备、存档、商店
- AI 生成、追击、寻路、同场战斗
- 场景边界、复杂碰撞、异常位置交互
- AI 梯度压测、自动化报告输出与回归

## 3. 执行环境

| 项目 | 说明 |
|---|---|
| 引擎版本 | Unreal Engine 5.7 |
| 测试平台 | Windows 11 |
| 测试地图 | `ActionRPG_P` |
| 输入方式 | 键鼠 |
| 自动化入口 | `run_rpg_automation.bat` |
| 关键人工步骤 | 进入场景后手动按一次 `BackSpace` |
