# C++ 测试扩展说明

## 改动原则

本次新增代码遵守以下边界：

- 不修改蓝图资源
- 不重写现有玩法逻辑
- 默认关闭，只有带参数时才启用

## 新增能力

### 1. 性能监控子系统

新增了独立的 `WorldSubsystem`，用于：

- 自动采样 FPS 和 FrameTime
- 记录 CPU、Actor、Pawn、Controller、AIController、AICharacter 数量
- 记录物理内存和虚拟内存占用
- 自动输出 `csv` 与 `summary.md`

### 2. 自动化测试子系统

新增自动化测试会话调度能力，用于：

- 启动一轮固定时长测试
- 自动执行控制台命令
- 自动输出 `autotest.md`
- 自动做轻量阈值判定

### 3. 关键链路性能埋点

针对以下链路补充了埋点：

- 背包增删改存读
- Ability 构建与应用
- Damage 执行
- Attribute 结算
- 角色能力初始化

## 价值

这部分 C++ 扩展的核心价值是：

- 提高项目可观测性
- 让测试结果沉淀成固定产物
- 让压测和回归不再只依赖手工观察
