# Jump 功能扩展说明

## 1. 扩展背景

原始 UE5 ActionRPG Demo 主要以地面战斗为主，角色移动能力相对有限。  
为验证角色移动能力变化后对玩法、地形可达性及 AI 行为的影响，我启用了 Jump 功能，并围绕其开展兼容性测试。

## 2. 扩展目标

该扩展主要用于支撑以下测试目标：

- 验证角色控制能力变化后的功能表现
- 观察高低差场景下的战斗行为
- 分析玩家移动范围变化对 AI 追击逻辑的影响
- 验证新增移动能力与原有攻击、落地逻辑之间的兼容性

## 3. 扩展意义

通过启用 Jump，测试范围从平面战斗进一步扩展到垂直空间相关场景，使角色控制与 AI 逻辑之间的耦合关系更容易被观察和验证。

该扩展使项目具备了更丰富的场景验证条件，也为兼容性测试提供了更有代表性的实验环境。
