# UE5 ActionRPG Demo 测试用例集

## 测试用例说明

本测试用例集为 **简化版本测试设计文档**，主要用于展示核心系统的测试思路与覆盖策略。

为了保证文档的可读性与维护成本，本用例集仅保留具有代表性的核心测试点，并未穷举所有边界情况。

本测试用例主要覆盖以下系统模块：

- 角色基础系统
- 战斗系统
- AI 行为系统
- 掉落与拾取系统
- 背包系统
- 商店系统
- 数据存储与恢复机制
- 性能与压力测试
- 灰盒专项验证

未详细展开的测试内容包括但不限于：

- UI 视觉细节验证
- 多分辨率与显示适配
- 音效与特效表现
- 网络与多人同步逻辑
- 输入设备兼容性

当前测试用例集侧重验证 **核心玩法逻辑、系统联动流程以及关键边界情况**。

在当前 Demo 项目规模下，本用例集可覆盖主要玩法逻辑与关键风险点，并保持较好的可读性与可维护性。
